Lightweight projection of the full events endpoint — returns only `name`, `coords`, `date_start`, `date_end` and `recurring`. Designed for rendering event pins on a map without the overhead of descriptions and full metadata.

Accepts the same query parameters as [`/pois/events`](/docs/poi-api/events).

---

## Query parameters

| Name | Type | Required | Default | Description |
|------|------|:--------:|---------|-------------|
| `mode` | string | — | `radius` | `radius` · `district` |
| `lat` | float | radius | — | Latitude WGS-84 |
| `lng` | float | radius | — | Longitude WGS-84 |
| `radius` | int | no | `5000` | Search radius in metres |
| `district` | string | district | — | District or city name |
| `lang` | string | no | `en` | ISO 639-1 language code |
| `limit` | int | no | `20` | Max results (max 100) |
| `offset` | int | no | `0` | Pagination offset |

---

## Response `200`

```json
{
  "total": 3,
  "results": [
    {
      "name":      "Fête de la Musique",
      "coords":    { "lat": 48.8566, "lng": 2.3522, "approximate": false },
      "recurring": true
    },
    {
      "name":       "Paris Jazz Festival",
      "coords":     { "lat": 48.8462, "lng": 2.2527, "approximate": false },
      "date_start": "2025-06-21",
      "date_end":   "2025-07-27",
      "recurring":  true
    }
  ]
}
```

### `SlimEvent` fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Event name |
| `coords` | object | `lat`, `lng`, `approximate` |
| `date_start` | string | ISO 8601 start date, omitted if unknown |
| `date_end` | string | ISO 8601 end date, omitted if unknown |
| `recurring` | bool | `true` for annually recurring events |

---

## Error responses

| Status | Condition |
|--------|-----------|
| `400` | Missing required parameters or unsupported mode |
| `429` | Rate limit exceeded |
| `500` | Internal server error |
